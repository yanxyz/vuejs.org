Class 与 Style 绑定

数据绑定一个常见需求是操作元素的 class 列表和它的内联样式。因为它们都是属性，我们可以用 v-bind 处理它们：我们只需要计算出表达式最终的字符串。不过，字符串拼接麻烦又易错。因此，在v-bind用于 class 和 style 时，Vue.js 专门增强了它。除了字符串，表达式的结果也可以是对象或数组。

绑定 HTML Class

尽管可以用 Mustache 标签绑定 class，比如 class="{{ className }}"，不推荐与 v-bind:class 一起用。两者只能选其一！
